## Build

### From zip file with dependencies included
- `#>` `mkdir build && cd build && cmake .. && make`
- `#>` `./particle_filter`

### From clean clone
- `#>` `mkdir build && cd build && cmake .. -DGIT_SUBMODULE=ON && make`
- `#>` `./particle_filter`


### Original scripts:

1. ./clean.sh
2. ./build.sh
3. ./run.sh
